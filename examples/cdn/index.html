ONNX Web Framework - CDN Usage Example

Note: This example demonstrates how to use ONNX Web Framework directly from CDN without any build tools.

Installation via CDN

Add the following script to your HTML file:

<!-- Load ONNX Runtime Web -->
<script src="https://cdn.jsdelivr.net/npm/onnxruntime-web@1.18.0/dist/ort.min.js"></script>

<!-- Load ONNX Web Framework -->
<script type="module">
  import ONNXWebFramework from 'https://cdn.jsdelivr.net/npm/onnx-web-framework@latest/dist/index.js';
</script>

Basic Usage

// Initialize framework
const framework = new ONNXWebFramework({
    executionProviders: ['wasm'],
    enableProfiling: true
});

await framework.initialize();

// Load model
await framework.loadModel('my-model', 'path/to/model.onnx');

// Run inference
const result = await framework.predict('my-model', inputData);

console.log('Inference result:', result.output);
console.log('Inference time:', result.inferenceTime);

Configuration Options

const framework = new ONNXWebFramework({
    // Execution providers (in priority order)
    executionProviders: ['webgpu', 'webgl', 'wasm'],

    // WebNN device type (cpu/gpu/npu)
    deviceType: 'gpu',

    // Enable performance profiling
    enableProfiling: true,

    // Worker configuration
    useWorker: true,

    // Caching options
    enableCache: true,
    cacheMaxAge: 24 * 60 * 60 * 1000, // 24 hours

    // WASM options
    numThreads: 4,

    // Debug mode
    debug: false,
    logLevel: 'warning'
});
